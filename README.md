# Espaço da Infosimples

Este repositório é um espaço dedicado a criadores de soluções de automação que usam a Infosimples.

Na área [Issues](https://github.com/infosimples/infosimples/issues) publicamos o que está na lista de desenvolvimento da
Infosimples e os serviços já disponibilizados, servindo para você acompanhar as novidades.

Em nosso [Histórico de mudanças](https://api.infosimples.com/mudancas) você encontra um registro com todas as alterações
significativas feitas na API da Infosimples.

## Suporte e Contato

Caso precise de ajuda ou queira fazer contato particular, basta enviar uma mensagem para:
**suporte@infosimples.com.br**

## Sobre a Infosimples

A Infosimples é uma empresa que disponibiliza centenas de [APIs para automação de consultas](https://infosimples.com/consultas/) em sites públicos e também [APIs de imagens (OCRs)](https://infosimples.com/imagens/).

* **Para criar sua conta gratuitamente:** [https://api.infosimples.com/cadastro](https://api.infosimples.com/cadastro)
* **Documentação de APIs de Consultas:** [https://api.infosimples.com/consultas/docs](https://api.infosimples.com/consultas/docs)
* **Documentação de APIs de Imagens:** [https://api.infosimples.com/imagens/docs](https://api.infosimples.com/imagens/docs)
